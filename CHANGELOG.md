# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

> New releases are synced here automatically via the [Sync Release to CHANGELOG](.github/workflows/sync-changelog.yml) workflow.

---

## [2026.419.0] - 2026-04-19

- Improved Git Integration.
- Git Diff.
- Added NPM and Stack Overflow.

---

## [2026.418.0] - 2026-04-18

- Added support for Git.
- MCP enhancements.
- Enhanced chat memory extraction.

---

## [2026.417.0] - 2026-04-17

- Removed the `rank` parameter from the models.
- Improved the steer option in chat.
- Improved knowledge support of the system.
- Improved web search.
- Added shortcuts.
- Improved chat history modal.
- Ability to pin 3 chats in history modal.

---

## [2026.416.0] - 2026-04-16

- Ability to steer the AI when its going out of track during a conversation.
- Improved System Prompt.
- Bug fixes.

---

## [2026.415.1] - 2026-04-15

- Added more tools during chat execution.
- Improved channel engine restart during errors.
- Added Channel messages to Events section, for better monitoring.

---

## [2026.415.0] - 2026-04-15

- Added support for Spotify, Sentry, Notion, Cloudflare, Vercel, Supabase, Netlify, Figma, Hubspot, Jira and Linear.
- Bug fixes.

---

## [2026.414.0] - 2026-04-14

- MCP preview rendering outside of chat session has been fixed.
- Under the hood changes.
- Bug fixes.

---

## [2026.413.2] - 2026-04-13

- The browser cache clearing issue has been fixed.

---

## [2026.413.1] - 2026-04-13

- Improved app loading.

---

## [2026.413.0] - 2026-04-13

- Faster response time.
- Improved support for Live Browser.
- Reduced Token usage.
- Smarter tools usage.

---

## [2026.412.0] - 2026-04-12

- Improved MCP support.
- Slightly faster boot.
- Bug fixes.
- Enhanced personal memory. (The more you chat, the more it learns about you)

---

## [2026.407.5] - 2026-04-07

- Added support for auto update.
- Bug fixes.

---

## [2026.407.0] - 2026-04-07

- Initial open source release.
